# DALL-E Image API
A simple web frontend to interact with OpenAI's DALL-E 3

## Running it

### Windows
1. Add your OpenAI API key to `run-user.ps1`
2. Run the script:
    ```
    pipenv shell
    .\run-user.ps1
    ```
